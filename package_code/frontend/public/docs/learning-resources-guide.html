学习资源与外链说明

本文档说明：学习仪表盘 → 知识点速览 → 查看详情 → 资源列表 中出现的链接为何指向外网、系统如何加载与打开链接，以及如何改为校内或自建资源地址。

1. 功能路径与界面行为

步骤位置操作
1学生端「学习仪表盘」在「知识点速览」区域看到若干知识点卡片。
2知识点卡片点击「查看详情」。
3知识点详情页（/student/kp/:id）展示掌握度、先修/后继与「资源列表」。
4资源卡片点击「开始使用」会在新标签页打开该资源的 URL。

资源数据来自数据库中的 url 字段；前端使用 window.open 打开，可为任意合法 HTTP(S) 地址或本站相对路径（如本页）。

2. 为何默认有「外网」链接

种子数据会为每个知识点生成示例文档与练习，其链接指向哔哩哔哩站内搜索，用于演示而无需在本项目内托管大量课件。若无法访问外网，这些链接可能无法打开。

本页为站内文档，不依赖外网，可在无公网环境下阅读说明。

3. 数据从哪里来

资源存储在表 Resource 中（kp_id、type、title、url 等）。学生端按知识点拉取列表并原样打开 url。

4. 如何改为内网或自有资源

  • 教师/管理员：通过资源 REST API 创建或更新资源，将 url 改为校内课件、LMS、对象存储等地址。
  • 开发环境：可修改 backend/seed.py 中生成资源的逻辑；注意重复执行种子与生产数据冲突问题。
  • 数据库：直接更新 resource 表的 url 列。

5. 使用与合规提示

  • 外网链接依赖网络策略；内网或离线环境请配置内网 url。
  • 外链版权与条款以第三方平台为准；正式开课建议使用已授权或自有资源。
  • 本项目前端未做域名白名单，录入链接时请做好审核。

7. 小结

资源是否为外网由数据库 url 决定；默认种子含哔哩哔哩搜索链接。本站说明文档路径为 /docs/learning-resources-guide.html，可作为不依赖外网的参考资源。

与仓库 docs/学习资源与外链说明.md 内容一致；静态页便于在资源列表中直接打开。
